Cone is a fast, fit, friendly, and safe
systems programming language


Design Goals

  • Fast: Optimal Performance
  • Friendly: Developer Agility
  • Safe: Memory, thread & type safe
  • Fit: Lean, native runtime

Distinctive Features:

  • Regions and Lifetimes: memory managed your way
  • Permissions and Actors: friendly concurrency
  • Variants, Structural Traits & Delegated Inheritance:
    sound polymorphic type system
