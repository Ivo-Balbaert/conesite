Cone is a fast, fit, friendly, and safe
systems programming language


Design Goals:

  • Fast, safe, "do-it-your-way" memory management
  • Efficient, safe concurrency
  • Versatile, sound type system
  • Concise, readable code
  • Memory, thread & type safe
  • Lean, native runtime

Ideal for complex, data-rich programs requiring
responsive performance (e.g., 3D)
