Polymorphic Reuse

Polymorphism is the ability to write some logic in a generic way and then have the compiler automatically specialize the logic to fit every type you apply to the generic code. Polymorphism improves programmer productivity by making it easy to create and use reusable code.

On the following pages, we will show how to

  • Create generic functions and types
  • Apply type equivalence constraints on the types a generic uses
  • Use traits to apply method and field constraints on a generic's types
  • Support runtime (virtual) dispatch using trait references
  • Create and use closure references
  • Inherit fields and methods
