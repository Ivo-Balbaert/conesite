Inheritance

Broadly speaking, inheritance is a mechanism that allows one type to reuse (inherit) the state or methods defined by some other type.

Inheritance is a namespace mechanism that enables a new type to "reuse" fields and methods defined by another type.

Imagine we have this type:

struct Node
  line u32
  col  u32

  fn printLoc() { ... }

This new type reuses all of Node's fields and methods:

struct BinaryNode
  node Node use *	
  left  u32
  right u32
	
  fn check() { ... }
