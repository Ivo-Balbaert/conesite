Introduction For C++ Developers

C++ is the workhorse programming language for 3D game development. Despite being over 30 years old, C++ gets regular updates that modernize and improve its features. C++ works, but its use still carries a significant productivity cost:

  • Include files. Every class requires two redundant files, wasting programmer time.
  • Build system. C++ package and dependency management are difficult and error-prone processes.
  • Memory and concurrency safety. C++ does not enforce reference safety, which means people waste time with extra testing, hard-to-diagnose bugs, and handling security vulnerabilities.
  • Complexity. The specification for C++ runs to hundreds of pages. This is a language that takes a long time to learn to use well.
  • Missing features. Many of the valuable features listed above are not supported by C++, particularly: variant types, built-in concurrency, slices, and tuples, as well as all Cone-exclusive features.
